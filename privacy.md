# Privacy Policy

**Last Updated:** 29-03-2026
**Bot Name:** Levelix

This Privacy Policy explains how Levelix collects and uses data.

## 1. Data We Collect

We may collect:

### a. Discord Data
- User IDs
- Server IDs
- Channel IDs
- Role IDs

### b. Message Data
- Message content (for leveling and moderation)
- Timestamps
- Command usage

### c. System Data
- Logs
- Errors
- Performance data

## 2. How We Use Data

We use data to:
- Provide bot features
- Track XP and levels
- Moderate servers
- Prevent abuse
- Improve the bot

Data is stored securely in databases and systems used to run the Bot.

We take reasonable measures to protect your data.

## 4. Data Sharing

- Sell your data
- Share data with advertisers

We may share data if required by law or to protect the bot.

Discord may collect its own data separately.

## 6. Message Content

Message content may be processed for:
- Leveling systems
- Anti-spam detection
- Logs

- Bot functionality

- Logs

We may delete data at any time.

## 9. User Rights
